📌 Overview

This project predicts the likelihood of heart disease using Machine Learning models trained on patient medical data. It is deployed as a live web application for real-time predictions.

🎯 Key Features
Real-time prediction using trained ML model
User-friendly web interface
End-to-end pipeline (data → model → deployment)
Accessible online via cloud deployment
🛠️ Tech Stack
Frontend: HTML / Streamlit
Backend: Python (Flask / Streamlit)
ML Libraries: Scikit-learn, Pandas, NumPy
Deployment: Render
⚙️ How It Works
User enters medical details (age, cholesterol, etc.)
Data is preprocessed (scaling, encoding)
Model predicts risk (0 or 1)
Result is displayed instantly
